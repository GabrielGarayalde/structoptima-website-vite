Timber Floor Framing Optimizer

The following program yields the optimal timber floor framing solution for a variable rectangular floor layout under a given set of loading parameters and design constraints. The program is based on the Monte-Carlo-Tree-Search algorithm and is capable of finding a near optimal solution in a matter of seconds.
